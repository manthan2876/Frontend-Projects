# Project 1: Single Page HTML CV

This is my solution for the [Single-Page CV project on roadmap.sh](https://roadmap.sh/projects/single-page-cv). It is built as part of a monorepo containing all my frontend engineering track projects.

## Project URL
https://roadmap.sh/projects/single-page-cv

## Tech Stack
- React
- Vite
- Tailwind CSS

## How to Run the Project Locally

1. **Navigate to the project directory:**
```bash
   cd html-cv

```

2. **Install dependencies:**

```bash
   npm install

```

3. **Start the local development server:**

```bash
   npm run dev

```

4. **Build for production:**

```bash
   npm run build

```
