# Project 8: Custom Image Grid Layout

This is my solution for the [Image Grid Layout project on roadmap.sh](https://roadmap.sh/projects/image-grid). It implements standard explicit cell spans inside responsive multi-dimensional canvas grids.

## Project URL
https://roadmap.sh/projects/image-grid

## Layout Architecture Features
- Built natively using **Tailwind CSS v4** layout engine frameworks and alignment controls.
- Features changing grid structure columns going fluidly from `grid-cols-1` on mobile screens up to `md:grid-cols-4` on wide viewports.
- Implements semantic `<figure>` and `<figcaption>` tracking containers for high-quality markup parsing.
