# Project 5: Testimonial Cards

This is my solution for the [Testimonial Cards project on roadmap.sh](https://roadmap.sh/projects/testimonial-cards). It highlights robust, multi-viewport card grid sizing models.

## Project URL
https://roadmap.sh/projects/testimonial-cards

## Features
- Scaled using **Tailwind CSS v4** with native `@tailwindcss/vite` compiler directives.
- CSS layout configurations accommodating alternating highlighted focal card elements seamlessly.
- Fully accessible semantic blockquote layout wrapping rules.
