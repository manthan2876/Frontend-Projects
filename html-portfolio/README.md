# Project 2: HTML Only Portfolio

This is my solution for the [HTML Only Portfolio project on roadmap.sh](https://roadmap.sh/projects/html-only-portfolio). It highlights core page architecture layout structural building blocks without visual styling logic.

## Project URL
https://roadmap.sh/projects/html-only-portfolio

## Features
- Multi-page context virtualization (Home, Projects, Articles, Contact)
- Clean semantic segment markers (`<header>`, `<nav>`, `<main>`, `<aside>`)
- Form control interfaces incorporating active type properties and labels
